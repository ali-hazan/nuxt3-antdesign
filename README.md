## Setup

Install dependency 

```bash
npm i
yarn
```

### Development Server

Start the development server on http://localhost:3000

```bash
yarn dev
npm dev
```

### Production Server

Regular Nuxt with SSR
```bash
yarn build
npm build
```

Prerendered SPA
```bash
yarn generate
npm generate
```

Storybook
```bash
yarn storybook
npm storybook
```
